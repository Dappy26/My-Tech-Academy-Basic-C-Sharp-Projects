# My-Tech-Academy-Basic-C-Sharp-Projects.
This repository contains various coding projects and exercises completed during my C# and .NET Framework course at The Tech Academy.
The projects included demonstrate a structural learning progression from fundamental object-oriented console applications to advanced multi-tier database configurations and full-stack Model-View-Controller (MVC) web architectures.

## 📁 Repository Directory & Project Categories

### 🌐 Full-Stack & Web Applications
* **Car Insurance MVC Application** – A complete ASP.NET MVC web application featuring a user-facing form that applies conditional risk-assessment business logic to dynamically calculate, display, and persist insurance quotes to a database backend. Includes a dedicated administrative view.
* **Server Time Razor App** – An ASP.NET Core Razor Pages application designed to pass dynamic backend server statistics and timestamps directly to front-end page layouts.

### 🗄️ Database Integration & ORM (Object-Relational Mapping)
* **Student Database Application** – A backend application utilizing an Entity Framework (EF6) Code-First workflow to programmatically design database schemas, configure data contexts via `DbContext`, and manage state persistence without writing raw SQL.
* **Database-First Management Modules** – Projects demonstrating database schema scaffolding, LINQ (Language Integrated Query) mapping, and structured CRUD operations interacting with local SQL Server instances.

### 💻 Object-Oriented Programming (OOP) & Core Logic
* **NFL Stats Parser** – An application utilizing foundational object-oriented principles, custom data structures, and algorithmic sorting logic to parse, organize, and filter athletic datasets via console file I/O operations.
* **Core C# Logic Labs & Challenges** – A collection of specialized console utilities focusing on system exception handling, array/list manipulations, string formatting, custom struct/enum creation, and multi-tier class inheritance.

## 🛠️ Technical Skills Architecture

* **Programming Concepts:** Object-Oriented Programming (OOP), Inheritance, Polymorphism, Encapsulation, Generics, Exception Handling.
* **Frameworks & UI:** .NET Framework, .NET Core, ASP.NET MVC, Razor Pages, Bootstrap, HTML5, CSS3.
* **Data Access & Storage:** Entity Framework (Code-First & Database-First), SQL Server, LocalDB, LINQ, File Streaming (CSV/JSON).
* **Workflows & Tools:** Visual Studio, Git Version Control, GitHub Repository Management.

## 🚀 Environment & Deployment Setup

Every subdirectory within this repository functions as an independent, standalone Visual Studio solution:
1. Clone this repository to your local computer environment.
2. Navigate into your target project directory.
3. Launch the respective `.sln` (Solution) file inside **Visual Studio** to build, compile, and execute the application locally.
